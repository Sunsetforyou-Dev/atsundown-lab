=== AtSundown FiveM ===
AtSundown เป็นร้านขายสคริปต์สำเร็จรูปสำหรับเซิร์ฟเวอร์ FiveM
ตอนนี้ AtSundown จำหน่ายเฉพาะสคริปต์สำเร็จรูปเท่านั้น
ตอนนี้ยังไม่รับงานเขียนสคริปต์ custom ไม่รับแก้ไขสคริปต์เดิม และไม่รับต่อยอดระบบเฉพาะ
ลูกค้าสามารถสั่งซื้อผ่านฟอร์มในหน้าเว็บ หรือคุยต่อทาง Discord ของ AtSundown

=== รายการสคริปต์และราคา ===
HUD Script ราคา 590 บาท
เหมาะสำหรับเซิร์ฟเวอร์ที่ต้องการหน้าจอสถานะผู้เล่น เช่น เลือด เกราะ อาหาร น้ำ และข้อมูลพื้นฐานของตัวละคร

Job Script ราคา 990 บาท
เหมาะสำหรับเพิ่มระบบอาชีพ งานฟาร์ม งานส่งของ งานตำรวจ งานหมอ หรืองานเฉพาะที่อยู่ในรูปแบบสคริปต์สำเร็จรูปของร้าน

Shop/Inventory Script ราคา 1,290 บาท
เหมาะสำหรับระบบร้านค้า คลังสินค้า ไอเทม หรือระบบซื้อขายที่เชื่อมกับ economy ของเซิร์ฟเวอร์

Gang/Activity Script ราคา 1,590 บาท
เหมาะสำหรับระบบแก๊ง กิจกรรม จุดยึดพื้นที่ ภารกิจ หรือระบบแข่งขันภายในเซิร์ฟเวอร์

=== วิธีสั่งซื้อ ===
ลูกค้าเลือกสคริปต์ที่ต้องการซื้อในฟอร์มสั่งซื้อบนหน้าเว็บ
ข้อมูลที่ควรกรอกคือ Discord สำหรับติดต่อกลับ สคริปต์ที่ต้องการซื้อ และรายละเอียดเซิร์ฟเวอร์หรือหมายเหตุ
หลังส่งออเดอร์แล้ว แอดมิน AtSundown จะตรวจสอบและติดต่อกลับทาง Discord
ระบบยังไม่รับชำระเงินออนไลน์ในหน้าเว็บ แอดมินจะปิดการขายและแจ้งขั้นตอนถัดไปทาง Discord

=== Framework ที่รองรับ ===
AtSundown รองรับงาน FiveM ทั่วไป และสามารถคุยรายละเอียดสำหรับ ESX หรือ QBCore ได้
ถ้าใช้ framework อื่น ให้แจ้งในหมายเหตุของออเดอร์เพื่อให้แอดมินตรวจสอบก่อนส่งมอบ

=== การติดตั้งและส่งมอบ ===
สคริปต์สำเร็จรูปจะส่งมอบตามช่องทางที่ตกลงกับแอดมินผ่าน Discord ของ AtSundown
รายละเอียดการติดตั้งขึ้นอยู่กับสคริปต์และสภาพเซิร์ฟเวอร์ของลูกค้า
ลูกค้าควรเตรียมข้อมูล server resource, framework, dependency และ error log ถ้ามีปัญหาระหว่างติดตั้ง

=== FAQ ===
Q: AtSundown ขายอะไร
A: AtSundown ขายสคริปต์สำเร็จรูปสำหรับ FiveM เท่านั้น

Q: มีสคริปต์ราคาเท่าไหร่บ้าง
A: HUD Script 590 บาท, Job Script 990 บาท, Shop/Inventory Script 1,290 บาท และ Gang/Activity Script 1,590 บาท

Q: รับเขียนสคริปต์ custom ไหม
A: ตอนนี้ยังไม่รับงานเขียนสคริปต์ custom จำหน่ายเฉพาะสคริปต์สำเร็จรูปเท่านั้น

Q: รับแก้ไขหรือต่อยอดสคริปต์เดิมไหม
A: ตอนนี้ยังไม่รับแก้ไขสคริปต์เดิมหรือรับต่อยอดระบบเฉพาะ

Q: ติดต่อร้านได้ทางไหน
A: ติดต่อผ่าน Discord ของ AtSundown หรือส่งออเดอร์ผ่านฟอร์มในหน้าเว็บ

Q: รับงาน ESX หรือ QBCore ไหม
A: สคริปต์ของร้านสามารถคุยรายละเอียดสำหรับ ESX และ QBCore ได้ ถ้าใช้ framework อื่นให้แจ้งในหมายเหตุของออเดอร์

Q: ราคานี้รวมติดตั้งไหม
A: รายละเอียดการติดตั้งให้คุยกับแอดมินผ่าน Discord ของ AtSundown เพราะขึ้นอยู่กับสภาพเซิร์ฟเวอร์และ dependency ที่ใช้
